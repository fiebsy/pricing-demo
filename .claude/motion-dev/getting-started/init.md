# Getting Started

Set up Motion in your React project and create your first animation.

---

## Topics

- [Installation](./installation.md): Install the Motion package and configure imports
- [Quick Start](./quick-start.md): Create your first animation with `<motion.div>`
- [Base UI Setup](./base-ui-setup.md): Integrate Motion with Base UI components using the render prop

---

## Quick Reference

| Task | Command/Code |
|------|--------------|
| Install | `npm install motion` |
| Import | `import { motion } from 'motion/react'` |
| Basic animation | `<motion.div animate={{ opacity: 1 }} />` |
| With Base UI | `<Component render={<motion.div />} />` |
