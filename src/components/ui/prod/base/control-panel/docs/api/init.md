## Quick Reference

| Type | Description |
|------|-------------|
| `Control` | Union type of all control variants |
| `Section` | Tab content with groups of controls |
| `PanelConfig` | Full panel configuration object |
| `ControlChangeEvent` | Emitted when any control value changes |
| `PanelContextValue` | Active tab and minimize state |
