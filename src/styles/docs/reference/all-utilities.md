## Shine Utilities

### Base Shine

| Class | Description |
|-------|-------------|
| `shine-none` | Remove shine |
| `shine-0` | 195° angle shine |
| `shine-0-subtle` | Subtle 195° |
| `shine-0-intense` | Intense 195° |
| `shine-1` | 180° shine |
| `shine-1-subtle` | Subtle 180° |
| `shine-1-intense` | Intense 180° |
| `shine-2` | Theme-aware shine |
| `shine-2-subtle` | Subtle theme-aware |
| `shine-2-intense` | Intense theme-aware |
| `shine-3` | Enhanced 2px shine |
| `shine-3-subtle` | Subtle enhanced |
| `shine-3-intense` | Intense enhanced |
| `shine-brand` | High contrast brand |
| `shine-brand-subtle` | Subtle brand |
| `shine-brand-intense` | Intense brand |

### Shine + Shadow

Pattern: `shine-{preset}-shadow-{size}` or `shine-{preset}-{intensity}-shadow-{size}`

Sizes: `xs`, `sm`, `md`, `lg`

Examples: `shine-1-shadow-md`, `shine-2-subtle-shadow-sm`, `shine-brand-intense-shadow-lg`
