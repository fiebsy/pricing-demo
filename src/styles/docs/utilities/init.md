# Utilities

Custom Tailwind utility classes for the PAYVA style system.

---

## Topics

- [Colors](./colors.md): Text, background, border, ring, and outline color utilities
- [Typography](./typography.md): Font families and display text sizes
- [Borders & Rings](./borders-rings.md): Focus states, decorative borders, outlines
- [Effects](./effects.md): Shine borders, depth gradients, animations, corners

---

## Quick Reference

### File Locations

| File | Purpose |
|------|---------|
| `utilities/colors.css` | Semantic color utilities |
| `utilities/typography.css` | Font and text utilities |
| `utilities/borders.css` | Border utilities |
| `utilities/rings.css` | Focus ring utilities |
| `utilities/outlines.css` | Outline utilities |
| `utilities/shine.css` | Shine border effects |
| `utilities/depth.css` | Depth gradient backgrounds |
| `utilities/gradients.css` | Gradient presets |
| `utilities/animations.css` | Animation classes |
| `utilities/corners.css` | Corner shape utilities |
| `utilities/misc.css` | Scrollbar, transitions |
| `utilities/silk-styles.css` | SILK component styles |
| `utilities/components-overrides.css` | Third-party overrides |

### Most Used Utilities

```tsx
// Colors
className="text-primary bg-secondary border-primary"

// Typography
className="font-display text-display-lg"

// Focus
className="ring-1 ring-brand"

// Effects
className="shine-1 depth-gradient-1"
```
